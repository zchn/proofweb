Login page of the ProofWeb interface for proof assistants

    The system offers a web interface for Coq, Isabelle, Matita (and incomplete for Lego, Plastic, HOL Light). Functionality is similar to those offered by ProofGeneral or CoqIde, but it works inside a web browser, without the need to install anything: no plugins, flush or java. It is developed to work with Gecko based browsers (Mozilla, Firefox, Galeon, ...), Opera (≥9), Safari (≥3) and partially IE (≥5.5; some features may not work, due to differences in browsers, there is little interest in IE, so they did not get fixed recently).
